# Discover Command

## Overview

The `/discover` command allows making players learn spells. This could be made for individual or all spells at once.

## Command Structure

The command has two main subcommands:

### Discover Spell

`/discover <player> <spell>` - Grants the specified player discovery of a single spell.

### Discover All

`/discover <player> all` - Grants the specified player discovery of all available spells.

> **Note:** Both subcommands require permission level 2 (operator).

## Examples

`/discover Steve ebwizardry:fireball` - Grants Steve discovery of the fireball spell

`/discover Alex all` - Grants Alex discovery of all spells she hasn't discovered yet
